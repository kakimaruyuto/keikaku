## カリキュラムの導入

### 1. 対象リポジトリをクローンする
```
git clone リポジトリURL
```

### 2. １で取得したリポジトリをVSCodeで開く
### 3. ブランチを切り替える
ブランチの切り替え方は２種類あります。
#### ① VSCodeで切り替える
左下に「main」と書いてある部分があるので、そこをクリックして作業ブランチを選択。
#### ② コマンドで切り替える
以下のコマンドで切り替えることができます。
```
git checkout ブランチ名
```
